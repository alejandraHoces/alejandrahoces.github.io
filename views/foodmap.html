Foodmap

"Foodmap es una web app la cual te permite filtrar los restaurantes que se encuentran cerca de ti.

Reto de código

El proyecto foodmap fue desarrollado en el Bootcamp de Laboratoria. El principal desafío fue crear una base de datos que contengan todos los restaurantes segun el plato a eleccion del usuario.
